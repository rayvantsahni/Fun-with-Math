# Sum of Sum of Divisors
For a number k, **σ(k)** denotes the **sum of all divisors of k**.

Given a number **n**, we need to return the **sum of all σ(i) from i = 1 to n**.

Example:

**σ(1)** = **1**

**σ(2)** = 1 + 2 = **3**

**σ(3)** = 1 + 3 = **4**

**σ(4)** = 1 + 2 + 4 = **7**

**σ(5)** = 1 + 5 = **6**

Hence,
For `n = 5`, we return 1 + 3 + 4 + 7 + 6 = `21`.
